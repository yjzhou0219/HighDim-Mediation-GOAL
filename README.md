# HighDim-Mediation-GOAL
Code for implementing the method that combines GOAL with an IPW-based semi-parametric estimator to estimate causal mediation effects for continuous exposures under high-dimensional covariates.


This project relies on the following R packages:

CRAN Packages: "remotes", "devtools", "stats", "MASS", "car", "glmnet", "dplyr";
GitHub Packages: They can be installed using devtools::install_github(). "sjstats" - devtools::install_github("strengejacke/sjstats"); "np" - install_github("JeffreyRacine/R-Package-np")

The code for this project was executed under R version 4.4.2 (2024-10-31).
